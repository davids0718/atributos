Navegadores web

Un navegador web (en inglés browser) es un software , aplicacion o programa que permite el acceso a la web, interprentando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.>.

La funcionalidad basica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos de multimedia incrustados

Ejemplos de navegadores web

Safari: Safari es un navegador web de codigo cerrado dasarrollado por Apple Inc. Está disponible para macOS, iPadOs e iOS.

Mozilla Firefox es un navegador web libre y de codigo abierto desarollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la corporacion Mozilla y la fundacion Mozilla.

Google Chrome es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de proyectos de codigo abierto.
